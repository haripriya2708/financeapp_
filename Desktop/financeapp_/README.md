Prerequisites
Node.js & npm
Python 3.9+
pip or conda

Run Locally
1. Clone the repo
git clone https://github.com/haripriya2708/financeapp_.git
cd financeapp_
2. Start Frontend
   cd client
   npm install
   npm start
3. Start the backend
   cd ../server
   pip install -r requirements.txt
   python app.py
4. Open http://localhost:3000
